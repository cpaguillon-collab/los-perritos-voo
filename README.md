# 🐶 Los Perritos — Camino a $100K

Dashboard interactivo de proyección de inversión en VOO (S&P 500).

**Camilo & Carlos** — Comprando 1 acción mensual de VOO a través de Trii (Colombia).

## 🚀 Ver online

👉 **https://TU-USUARIO.github.io/los-perritos-voo/**

## 📊 Qué incluye

- Métricas actuales del portafolio
- Barra de progreso hacia $100K
- Gráfica de crecimiento con interés compuesto
- 4 escenarios de retorno (8%, 10.5%, 12%, 14.6%)
- Hitos del camino ($10K → $25K → $50K → $75K → $100K)
- Controles interactivos para simular diferentes escenarios
- Comparación: ahorro simple vs. interés compuesto

## 🛠 Tecnologías

- HTML/CSS/JS vanilla
- Chart.js 4.4.1
- Google Fonts (DM Sans + Instrument Serif)
